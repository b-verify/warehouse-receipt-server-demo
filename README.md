# b\_verify DEMO!

Simple demo of the b\_verify protocol. 

There are 4 components for the demo: the server, Alice (a depositor), Bob (a depositor) and Warehouse. Run each component in a separate window using the following scripts

```./run_server.sh```
```./run_alice.sh```
```./run_bob.sh```
```./run_warehouse.sh```
